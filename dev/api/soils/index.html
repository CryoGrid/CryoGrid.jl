Soils

CryoGrid.Soils.HomogeneousMixture — Type
HomogeneousMixture{Tpor,Tsat,Torg} <: SoilParameterization

Represents a simple, uniform organic/mineral soil mixutre in terms of its characteristic fractions: i.e. natural porosity, saturation, organic solid fraction, and excess ice fraction.

source
CryoGrid.Soils.HomogeneousSoil — Type
HomogeneousSoil{Tpara<:SoilParameterization,Tprop,Tsp,TP} <: SubSurface{TP}

Generic, homogeneous Soil layer, i.e. material is assumed to be uniformly mixed.

source
CryoGrid.Soils.MineralSediment — Type
MineralSediment{Tpor,Tsat} <: SoilParameterization

Represents a simple, homogeneous, mineral sediment with (possibly) spatially variable porosity.

source

Soil(

soilproperties(para::SoilParameterization, proc::Process; prop_kwargs...)

Constructs a default set of soil properties/constants based on the given parameterization and process(es). The default implementation simply constructs a NamedTuple from the given keyword arguments.
