CryoGrid.Physics.HeatConduction.FreeWater — Method
(fc::FreeWater)(sub::SubSurface, heat::Heat{FreeWater,Enthalpy}, state)

Implementation of "free water" freeze curve for any subsurface layer. Assumes that 'state' contains at least temperature (T), enthalpy (H), heat capacity (C), total water content (θw), and liquid water content (θl).

CryoGrid.Physics.HeatConduction.heatconduction! — Method
heatconduction!(∂H,T,ΔT,k,Δk)

1-D heat conduction/diffusion given T, k, and their deltas. Resulting enthalpy gradient is stored in ∂H. Note that this function does not perform bounds checking. It is up to the user to ensure that all variables are arrays of the correct length.
